# Collections Print Report - Quick Start Guide

## For End Users

### Generating a Collections Report

1. **Navigate to Collections Management**

    ```
    Dashboard → Financial → Daily Fee Collections
    ```

2. **Filter Your Data** (Optional but Recommended)

    - **Site:** Select specific campus/site or leave blank for all sites
    - **Academic Year:** Choose year (e.g., "2024/2025")
    - **Term:** Select term (1, 2, or 3) or leave blank for all terms
    - **Date Range:** Set "From" and "To" dates for specific period
    - Click **"Apply Filters"** button

3. **Generate the Report**

    - Click the **"Print Report"** button (blue button with printer icon)
    - Browser print dialog opens automatically
    - Preview the report in the print dialog

4. **Print or Save**
    - **To Print:** Click "Print" and select your printer
    - **To Save as PDF:**
        - Change destination to "Save as PDF"
        - Click "Save"
        - Choose location and filename

### Tips for Best Results

-   **Landscape vs Portrait:** Use Portrait orientation for best fit
-   **Margins:** Use default margins (15mm) for optimal layout
-   **Background Graphics:** Enable "Background graphics" in print settings for colored headers
-   **Scale:** Use 100% scale (do not shrink or expand)
-   **Color vs B&W:** Report is designed for both color and grayscale printing

## For Administrators

### Customizing School Information

The report automatically pulls:

-   School name from your profile
-   User information for "Generated By" field
-   Academic period from your filters

To update permanent school details:

1. Edit `/components/DailyFeeCollectionManagement.tsx`
2. Find the hidden print component (near the end of the file)
3. Update these props:
    ```typescript
    schoolAddress = 'Your School Address Here';
    schoolContact = 'Tel: +233 XXX XXX XXX | Email: contact@school.edu.gh';
    schoolLogo = '/path/to/logo.png'; // Optional
    ```

### Report Content Breakdown

The printed report includes:

#### Header Section

-   School logo (if configured)
-   School name (large, bold)
-   School address and contact
-   Report title: "COLLECTIONS REPORT"
-   Academic year and term
-   Site name (if filtered)
-   Date range

#### Summary Cards

-   **Total Collections:** Grand total of all fees collected
-   **Total Transactions:** Number of collection records
-   **Average Daily:** Average collection per day

#### Collections Table

Columns:

1. No. (Sequential)
2. Date (DD MMM YYYY format)
3. Site
4. Attendance (Present/Total + %)
5. Canteen Fee
6. Bus Fee
7. Total (Canteen + Bus)

Footer row shows totals for all monetary columns.

#### Signature Section

Three signature lines for:

-   Prepared By
-   Verified By
-   Approved By

#### Footer

-   Generated by: [User Name]
-   Date & Time: [Generation timestamp]

## Common Use Cases

### 1. Weekly Financial Review

```
Filters:
- Date From: Start of week (e.g., Monday)
- Date To: End of week (e.g., Friday)
- Click "Apply Filters" → "Print Report"
```

### 2. Monthly Report for Specific Site

```
Filters:
- Site: [Select campus]
- Date From: 1st of month
- Date To: Last day of month
- Click "Apply Filters" → "Print Report"
```

### 3. Term Summary Report

```
Filters:
- Academic Year: [Current year]
- Term: [Select 1, 2, or 3]
- Leave dates blank for entire term
- Click "Apply Filters" → "Print Report"
```

### 4. Annual Collections Report

```
Filters:
- Academic Year: [Select year]
- Leave term and dates blank
- Click "Apply Filters" → "Print Report"
```

### 5. All Collections (No Filter)

```
- Leave all filters blank
- Click "Print Report" directly
- Shows all collection records
```

## Keyboard Shortcuts

When print dialog is open:

-   **Ctrl+P** / **Cmd+P:** Open print dialog
-   **Enter:** Confirm print
-   **Esc:** Cancel print dialog

## Troubleshooting Quick Fixes

| Problem                             | Solution                                                               |
| ----------------------------------- | ---------------------------------------------------------------------- |
| Report is empty                     | Ensure you have collection data. Check if filters are too restrictive. |
| "Print Report" button is grayed out | No data available. Adjust filters or add collection records first.     |
| Report doesn't fit on page          | Check print scale is 100%, not "Fit to page"                           |
| Missing colors in print             | Enable "Background graphics" in print settings                         |
| Wrong school name showing           | Contact administrator to update school information                     |
| Currency symbol incorrect           | Verify currency setting in collection records                          |

## Professional Presentation Tips

1. **Consistent Branding**

    - Use school logo on reports
    - Ensure school contact information is current
    - Use official school colors (if customizing)

2. **Clear Date Ranges**

    - Always specify date ranges in filters
    - Use full terms or complete months when possible
    - Document report period clearly

3. **Signature Protocol**

    - Print report
    - Have appropriate personnel sign in designated areas
    - Scan or photocopy for records if needed

4. **Filing Recommendations**
    - Save PDF copy digitally
    - Print physical copy for audit files
    - Include date generated in filename
    - Example: `Collections_Report_2024-01-15_to_2024-01-31.pdf`

## Report Validation Checklist

Before finalizing a report, verify:

-   [ ] Correct date range
-   [ ] Correct academic year and term
-   [ ] All collection entries are visible
-   [ ] Totals are accurate
-   [ ] School information is current
-   [ ] "Generated by" shows correct user
-   [ ] Print preview looks professional
-   [ ] No data is cut off or clipped

## Export Options

### Save as PDF (Recommended for Digital Records)

1. Click "Print Report"
2. In print dialog, select "Save as PDF"
3. Click "Save"
4. Name file descriptively: `Collections_[DateRange].pdf`
5. Store in appropriate folder

### Email Report

1. Generate and save as PDF (see above)
2. Open email client
3. Attach the PDF file
4. Send to relevant stakeholders

### Physical Copy for Audit

1. Click "Print Report"
2. Select physical printer
3. Use high-quality paper
4. Print in color (recommended)
5. Have appropriate personnel sign
6. File in audit binder
