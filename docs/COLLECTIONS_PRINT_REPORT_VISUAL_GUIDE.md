## Responsive Considerations

### For Smaller Datasets (<20 records)

-   All content fits on single page
-   More whitespace between sections
-   Larger signature section

### For Larger Datasets (>100 records)

-   Table spans multiple pages
-   Headers repeat on each page (future enhancement)
-   Footer only on last page
