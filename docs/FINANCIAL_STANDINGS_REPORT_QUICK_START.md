# Financial Standings Report - Quick Start Guide

## For End Users

### How to Generate a Financial Report

#### Step 1: Access Dashboard

1. Navigate to **Financial Controller Dashboard**
2. Wait for financial data to load

#### Step 2: Apply Filters (Optional)

You can filter the report data by:

-   **School**: Select specific school or leave blank for all schools
-   **Site**: Choose a specific campus/location
-   **Academic Year**: Select the academic year (e.g., 2024/2025)
-   **Period View**: Choose time period
    -   Today
    -   This Week
    -   This Month
    -   This Term
    -   This Year
-   **Date Range**: Set custom start and end dates

> **Note**: Applied filters will be shown on the printed report for transparency

#### Step 3: Generate PDF Report

1. Click the **"Export PDF"** button in the top toolbar
2. Browser print dialog will open automatically
3. Configure print settings:
    - **Destination**: Choose "Save as PDF" or select a printer
    - **Layout**: Portrait (recommended)
    - **Pages**: All
    - **Color**: Color (recommended)
    - **Margins**: Default
    - **Options**: ‚úÖ Check "Background graphics"
4. Click **Print** or **Save**
5. PDF will be downloaded or sent to printer

#### Step 4: Success Confirmation

-   A green success notification will appear
-   PDF file is saved with name: `Financial_Standings_Report_YYYY-MM-DD.pdf`

## Troubleshooting

### PDF Not Generating

**Problem**: Print dialog doesn't open  
**Solution**:

-   Check browser pop-up blocker settings
-   Allow pop-ups for this site
-   Try different browser (Chrome recommended)

### Colors Missing in PDF

**Problem**: Report appears black and white  
**Solution**:

-   Enable "Background graphics" in print dialog
-   Check "Print backgrounds" option
-   Use "Save as PDF" option instead of print

### Data Appears Incorrect

**Problem**: Numbers don't match dashboard  
**Solution**:

-   Refresh the page to reload latest data
-   Check filters - they affect both dashboard and report
-   Verify the period view selection

### Report Cuts Off

**Problem**: Content is clipped or missing  
**Solution**:

-   Use default margins
-   Select "Fit to page" if available
-   Check paper size is set to A4

### Logo Not Showing

**Problem**: School logo missing  
**Solution**:

-   Ensure logo is uploaded in school settings
-   Check logo file format (PNG/JPG recommended)
-   Verify logo URL is accessible
