# Unified Education Platform â€” UI Design System

## Executive Summary

This document specifies a modern, unified UI for the integrated School Management System (SMS) + Learning Management System (LMS). The design treats the platform as **one cohesive education product** rather than two merged systems.

---

## 1. Design Philosophy

### 1.1 Core Principles

| Principle                       | Description                                                                                |
| ------------------------------- | ------------------------------------------------------------------------------------------ |
| **Unified Experience**          | One platform, one design language â€” no visual distinction between "SMS" and "LMS" features |
| **Role-Adaptive**               | UI morphs based on user role while maintaining consistent patterns                         |
| **Progressive Disclosure**      | Show only what's needed; reveal complexity gradually                                       |
| **Mobile-First Responsive**     | Touch-friendly, works on all devices                                                       |
| **Accessibility (WCAG 2.1 AA)** | Keyboard navigable, screen reader compatible, high contrast                                |
| **Performance First**           | Skeleton loaders, optimistic updates, lazy loading                                         |

## 9. Accessibility Guidelines

### 9.1 Keyboard Navigation

| Key               | Action                         |
| ----------------- | ------------------------------ |
| `Tab`             | Move focus forward             |
| `Shift + Tab`     | Move focus backward            |
| `Enter` / `Space` | Activate focused element       |
| `Escape`          | Close modal / cancel           |
| `Arrow Keys`      | Navigate within menus / tables |

### 9.2 Focus Management

```scss
// Focus ring style
*:focus-visible {
    outline: 2px solid $primary-500;
    outline-offset: 2px;
}

// Skip to content link
.skip-link {
    position: absolute;
    top: -40px;
    left: 0;
    &:focus {
        top: 0;
    }
}
```

### 9.3 Color Contrast

-   Text on backgrounds: Minimum 4.5:1 ratio
-   Large text (18px+): Minimum 3:1 ratio
-   UI components: Minimum 3:1 ratio

### 9.4 ARIA Labels

```tsx
// Icon-only buttons
<Button aria-label="Delete course" icon={<TrashIcon />} />

// Progress indicators
<ProgressBar
  value={65}
  aria-label="Course completion progress"
  aria-valuenow={65}
  aria-valuemin={0}
  aria-valuemax={100}
/>

// Live regions for dynamic content
<div aria-live="polite" aria-atomic="true">
  {statusMessage}
</div>
```
